# 前端轻量美化计划

## 目标
保持单文件 HTML 架构，通过 CDN 引入 Tailwind + 轻量优化提升 UI 体验。

## 技术方案
- **CSS**: 通过 CDN 引入 Tailwind Play CDN（无需构建）
- **JS**: 保持原生 JavaScript，不引入框架
- **文件**: 修改 `src/pypitravel/static/index.html` + `src/pypitravel/cli.py`

## 实施步骤

### 1. 后端：缓存列表 API 返回行程名称
修改 `/api/cached-journeys`，读取每个缓存文件提取行程名称：
```python
# 返回格式改为
{
  "journeys": [
    {"id": "123456", "name": "日本7日游"},
    {"id": "789012", "name": "泰国5日游"}
  ]
}
```

### 2. 引入 Tailwind CDN
```html
<script src="https://cdn.tailwindcss.com"></script>
```

### 3. 页面层次结构设计
```
┌─────────────────────────────────────────┐
│  PyPitravel - 圆周旅迹行程规划          │  ← 顶部标题栏
├─────────────────────────────────────────┤
│  ┌─────────────┐  ┌─────────────────┐   │
│  │ 缓存行程    │  │ 解析新行程      │   │  ← 两个并排卡片
│  │ [下拉选择]  │  │ [输入框]        │   │
│  │ [加载]      │  │ [解析]          │   │
│  └─────────────┘  └─────────────────┘   │
├─────────────────────────────────────────┤
│  📋 行程详情 - 日本7日游    [导出CSV]   │  ← 表格区域
│  ┌─────────────────────────────────────┐│
│  │ 日期/计划 │ 事件内容                ││
│  ├───────────┼─────────────────────────┤│
│  │ Day 1     │ ...                     ││
│  └─────────────────────────────────────┘│
└─────────────────────────────────────────┘
```

### 4. 缓存列表优化
- 下拉选项显示格式：`行程名称 | ID`（如：`日本7日游 | 123456`）
- 加载缓存后表格标题显示行程名称

### 5. 输入框优化
- placeholder: `粘贴行程链接或输入ID，如：123456`
- 输入框宽度加大，支持粘贴长链接

### 6. 交互增强
- 按钮添加 loading 状态
- 错误提示改为 toast 样式

## 修改文件
1. `src/pypitravel/cli.py` - 修改 `/api/cached-journeys` 返回名称
2. `src/pypitravel/static/index.html` - 页面美化和交互优化

## 验证方式
1. 运行 `uv run pypitravel`
2. 测试缓存列表显示行程名称
3. 测试输入 ID 和链接都能正常解析
4. 检查响应式布局
